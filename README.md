# Docker WordPress Develpor

基于 Docker 环境，安装 WordPress，并且创建一个简单的主题。

### 开始

```
cd wordpress
```

```
docker-compose up
```

### 目录说明

```
./
├── LICENSE
├── README.md
└── wordpress
    ├── devpoint  // 简单的主题
    │   ├── index.php
    │   └── style.css
    └── docker-compose.yml
```
